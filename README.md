# 14 laboratory


#### Run this command to build jar and run tests: 

```
mvn clean compile assembly:single
```

#### Run this to run jar:
__Windows__
```
java -jar target/lab14-1.0-SNAPSHOT-jar-with-dependencies.jar
```
__Linux__
```
java -jar ./target/lab14-1.0-SNAPSHOT-jar-with-dependencies.jar
```
